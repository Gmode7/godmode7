## Skill Progression

### Junior QA (Level 1)
- Executes test cases
- Reports bugs clearly
- Runs automated tests
- Follows test plans

### Mid QA (Level 3)
- Designs test cases
- Analyzes requirements
- Creates test plans
- Basic automation
- Identifies coverage gaps

### Senior QA (Level 5)
- Defines test strategies
- Builds automation frameworks
- Performance testing
- Mentors junior QAs
- Influences quality culture

### Staff QA (Level 7)
- Sets organizational standards
- Quality architecture
- Test infrastructure design
- Release quality ownership
- Coaches teams
